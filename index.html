  <h1>Privacy Policy</h1>

  <p><strong>Last updated:</strong> April 8, 2025</p>


  <p><strong>AI Resume Revise</strong> is designed with user privacy in mind. This policy explains how data is handled within the app.</p>


  <h2>Data Collection</h2>

  <p>This app does <strong>not collect, store, or share</strong> any personal data. No user accounts, login credentials, contact information, or location data are required or stored.</p>


  <h2>Resume Processing via DeepSeek API</h2>

  <p>Resume and job description inputs are sent to the <strong>DeepSeek API</strong> for real-time processing. Data is used solely to generate a response. No information is stored or logged by this app.</p>

  <p>DeepSeek’s privacy practices apply to this processing. See their <a href="https://www.deepseek.com/#/policy" target="_blank">Privacy Policy</a>.</p>


  <h2>Local Data Storage</h2>

  <p>All revision history or generated results are stored <strong>locally on the user’s device</strong> using SwiftUI’s default storage mechanisms. This data is not transmitted externally. Deleting the app will remove this data.</p>


  <h2>Security</h2>

  <p>All communication with the DeepSeek API is secured via <strong>HTTPS</strong> encryption to protect user input during transmission.</p>


  <h2>Third-Party Services</h2>

  <p>This app does <strong>not</strong> include third-party SDKs for analytics, advertising, or user tracking.</p>


  <h2>Policy Updates</h2>

  <p>Any changes to this privacy policy will be updated on this page, with a revised "Last updated" date.</p>
